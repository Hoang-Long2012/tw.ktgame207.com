## Game Introduction:

**Name:** TW (Terra Wilds

**Genre:** Survival, Open World.

**Developer:** Technology Entertainment Team.

**Homepage:** [official website of tw](https://tw.ktgame207.com/)

## Game Overview:

TW (Terra Wilds) is a combination of daily life, adventure, and endless challenges.

In this game, you can experience everything from being an explorer to a skilled craftsman if you have a solid knowledge of the map, or simply a normal fisherman earning a living day by day.

Everything depends on your own decisions.

TW features a diverse system of attributes such as hunger, thirst, dirtiness, illness, and many other factors.

### Key Highlights:

* **Lucky Spin System:** This system allows everyone to spin daily gifts and win items, levels, or random rewards.
* **Map System:** The game features many vast maps for players to explore and adventure.
* **Item System:** There are many interesting and rare items waiting for you to hunt and discover.
* **Hunt System:** Of course, a survival game must have a hunting system so players can kill animals, slaughter them, and eat them to survive day by day.
* And many other features.

## First-Time Startup:

When you launch the game for the first time, you will be greeted by a welcome screen. Press **Enter** to skip it.

After that, an agreement dialog accepting the game's rules will appear. If you agree to the rules of the game, you can press **Tab** to navigate to `I accept all terms and conditions that I have read` (the shortcut for this is **Alt + A**), and then press **Space** to check it.

After checking `I accept all terms and conditions that I have read`, press **Tab** to navigate to `Continue` (shortcut: **Alt + C**).

*Note: You can read the rules of the game again if you want in `Current_Game_Folder\documentation\rules.txt`.*

Next, you will have to pass through a few screens thanking you for agreeing to the rules, choosing to play the game, and respecting the admins regarding feedback and bug reports.

You can skip these by pressing **Enter**.

Afterward, the game will ask you to select a language. Currently, the game only supports English, Vietnamese, and Turkish; more languages may be supported in the future.

You can contribute more translations for the game via the information provided at the bottom of the page.

The default language is English.

In this guide, I will assume the language you are using is English by default.

After that, the game will start playing the intro logo. You can skip it by pressing **Enter**.

*Note: You will hear this logo sound every time you launch the game unless you disable the `Play logo on startup` option in the game settings.*

After the logo sound ends, you will be brought to the main menu.

* **Redownload menu:** This option is used to redownload game files if you accidentally deleted a game file or if the files are corrupted.
* **Restart your client:** Click here to restart your game client.
* **Exit:** This option is used to exit the game.

## configure option:

In this section you can customize according to your preferences because they are systematically arranged. you can access by pressing enter at **configure option** 

### General tab:

will have the following options:

* **Ask for confirmation on exit:** If you check this option, the game will ask for confirmation every time you go offline/exit.
* **Play logo on startup:** If checked, the game will play the logo sound every time it starts up.
* **Automatically check for updates on startup:** If checked, the game will automatically check for updates upon startup.
* **Silence screen reader alerts when window is inactive:** If checked, the game will stop sending notifications to the screen reader when the game window is inactive/out of focus.
* **Buffer logging:** When checked, the game will record your buffer log. This is a developer-only option.
* **reset buffer positions:** will restore buffer everything back to the way it was.
* **autologin on startup:** You will be taken directly to the server after running the game and skip the logo.
* **Read online and offline messages and alerts:** If checked, the server will notify you every time another player goes online or offline.
* **Level up alerts:** If checked, the server will notify you when you or another player levels up.
* **Dead messages and alerts:** If checked, you will receive notifications from the server when a player dies.

### Sound tab:

Here allows you to customize the game's sound to best suit you:

* **Thicken the pitch of sounds on exit:** If checked, the game will play a fading-out/gradually lowering sound when you go offline.
* **enable connection beeps:** When this option is enabled a beep will sound if you are disconnected from the server or you have to wait too long if you activate the login from the main menu.
* **Jumping Sound List:** You can adjust the jump sound during gameplay. There are many different sounds for you to choose from.
* **menu sound theme List:** If you have a favorite menu sound, you can adjust it here.
* **pan menu sounds left to right based on position:** When this option is enabled, your menu sound will move from left to right in the main menu.
* **wrap around at the ends of menus:** Now you can safely go to the bottom or top of the menu without any serious problems.
* **use left and right arrows to move through menus:** You can rotate the menu from vertical to horizontal.

### Voice Chat tab:

Here are the statistics about the voice chat system:

* **save received voice chats to disk (.opus files in voices folder):** Voice files will always be saved in the game directory.
* **Microphone samplerate:** This option determines the quality of your voice messages.
* **Microphone channels:** You can select your microphone channel from this option.
* **voice chat enabled:** When this option is enabled your voice will be enabled.
* **hear my own voice while I speak:** You can hear your own voice while chatting with other players.
* **hear other players:** This option helps nearby players hear your voice.
* **speak mic on/off when voice chat starts and stops:** This option helps your microphone always turn on and off properly.
* **microphone (capture device) List:** allows you to adjust the device for the microphone.
* **playback device (speakers / headphones) List:** You can adjust the speaker's settings to hear more clearly.
* **voice quality preset List:** You can customize many different voice levels to suit your needs.
* **manage custom presets:** You can manage the voice level customization system here.
* **advanced voice settings:** This is an advanced section for those who are interested in using it with many other extremely interesting modes.
  * noise gate (suppress quiet background noise): allows you to completely eliminate surrounding noise.
  * automatic gain control (level out my voice): Helps you balance your voice at any time.
  * voice activity detection (don't send when silent): When you turn off voice chat, this option will help you prevent accidental sending of voice messages.
  * forward error correction (better packet loss handling): helps prevent voice conversion errors.
  * packet loss concealment (synthesise lost frames): This is probably quite a technically complicated option and it won't be explained here.
  * reset to defaults: You can restore them to their original state if you feel you made a mistake in any step.

Finally, the remaining two buttons are for saving configuration and canceling if you don't want to cause errors.

## Creating a New Account:

You can create a new account using the `Create account` option in the main menu.

After clicking `Create account`, a dialog will appear asking you to enter a username, password, email, select your character's gender, and accept the game rules.

You can navigate through these fields using **Tab** and **Shift + Tab**.

* **Username Field:** Enter your username (the name you want to give to your character) with no spaces. (Shortcut: **Alt + U**).
* **Password Field:** Enter your password. To prevent unauthorized access, make sure your password is at least 4 characters long and combines uppercase letters, lowercase letters, numbers, punctuation marks, and other special characters. (Shortcut: **Alt + P**).
* **Email Field:** Enter your email address. *Note: If you use a fake/invalid email address and forget your password, you will not be able to recover your account.* (Shortcut: **Alt + E**).
* **Gender Selection:** The game will ask you to choose a gender for your character (Male or Female). You can select using the **Up** and **Down Arrow** keys.
* **Rule Acceptance Checkbox:** Next, you will see a checkbox: `I have read and accept the rules of the game, and I accept the punishment by the administrators If I'll do something wrong. I will never use any bugs, but instead, will report them as soon as I found any. also I accept to punish me If I have used any bugs for personal gain instead of reporting them, as well for the game improvement.` (Shortcut: **Alt + A**). To check it, press **Space**.
* **Create account Button:** After agreeing to the rules, you can **Tab** to the `Create account` button and press **Enter** to instantly create a new account. (Shortcut: **Alt + C**).
* **Cancel Button:** If you do not wish to create a new account, you can **Tab** to the `Cancel` button and press **Enter** to cancel. (Shortcut: **Alt + C**).

*Notes:*
* You can quickly trigger the `Create account` button by pressing **Shift + Enter**.
* You can quickly cancel creating a new account by pressing **Escape** within the creation dialog.
* After clicking `Create account`, if an error occurs during the process, the game will notify you. You can press **Enter** to close the notification. Afterward, the game will return you to the main menu.
* If the account is successfully created, the game will show a message: `Success! Account created! Press enter to decide whether login or not.` Press **Enter** to dismiss this message.
* After dismissing the success message, the game will ask: `Do you want to login to the server now?`
  * Select `Yes, log me in!` to log in immediately.
  * Select `No, not right now, will login later!` to return to the main menu.

## Logging into an Existing Account:

If you already have an account, you can log in using the `Setup an existing account` option in the main menu.

After clicking `Setup an existing account`, a dialog will appear asking you to enter the username and password of the account you want to log into.

You can navigate through these fields using **Tab** and **Shift + Tab**.

* **Username Field:** Enter your username (Shortcut: **Alt + N**).
* **Password Field:** Enter your password (Shortcut: **Alt + P**).
* **Setup Account Button:** After entering your username and password, press **Tab** to the `Setup account` button and press **Enter** to log in. (Shortcut: **Alt + S**).
* **Cancel Button:** If you do not want to proceed with logging in, you can **Tab** to the `Cancel` button and press **Enter** to cancel. (Shortcut: **Alt + C**).

After clicking `Setup account`, you will see a notification: `Account information updated successfully.` Press **Enter** to dismiss it, and you will be returned to the main menu.

## First-Time Login:

You can connect to the server using the `Login as User_Name` option in the main menu if you have already registered a new account or logged into a previously created account.

The first time you log into a new account, the game will ask you to select a help language: `Please select a language. Help or something will be shown to you in the language you choose`. Currently, the game only supports English and Turkish; more languages may be supported in the future.

After selecting the help language, you will be brought into the **Safe Zone**, carrying a `startup_pack` (starter pack) and a `wallet` in your inventory.

## Basic In-Game Information:

This section covers basic in-game stats, states, and the two gameplay modes (Item Mode and Skill Mode).

### Basic Stats:

* **Health (H):** If your health drops to 0, you die.
  * By default, each player has 25,000 health.
  * Your health will decrease when you are attacked.
  * To restore health after combat, you can drink milk, use a `health_booster`, or sit still.
  * You can increase your maximum health limit by consuming certain foods and drinks (such as Soda and Cola) or by increasing your stars.
* **Energy (E):** If your energy drops too low, your character will begin to yawn, feel sleepy, walk slower, stumble, and potentially fall down at any time.
  * If your energy drops below 0, your character will automatically fall asleep to recover energy.
  * Besides sleeping and sitting still, you can recover energy by using an `energy_drug` or eating chocolate.
  * Certain actions that require more effort than normal (such as riding a bike or rowing a boat) will cause your energy to drain faster.
  * By default, each player has 1,000 energy.
* **Hunger (U):** If your hunger level reaches 1,000, you die. You can reduce hunger by eating food.
* **Thirst (S):** If your thirst level reaches 1,000, you die. You can reduce thirst by drinking water/beverages.
  * *Note: In certain areas like the desert, your thirst level will increase much faster.*
* **Level (L):** Your level determines your rank/title in the game.
* **Stars:** Approximately every 4,000 levels, you will earn 1 star. If you have too many stars, you will not be able to return to the Safe Zone.
  * If your star count is a negative number, it will automatically reset to 1.
  * Besides leveling up, you can also earn stars from the daily lucky spin.
  * *Warning: The lucky spin may sometimes deduct some of your stars.*
  * When you gain a star, your maximum health will increase slightly.

*Note: A warning sound will play when your hunger or thirst levels reach 800 or above. You can check your detailed stats and info by pressing **Shift + S**.*

### Peace & War States:

The game features two main states: Peace and War. You can toggle your state using **Shift + F2**.

* **Peace State:** You cannot attack anything, and you cannot reload ammunition into guns.
* **War State:** Opposite to Peace State; allows you to fight, attack, and reload your weapons.

### Map & Movement:

TW is a survival audio game featuring both 3D and 2D map systems, utilizing coordinates: **x** (Width), **y** (Length), and **z** (Height).

* The game primarily uses 3D maps, giving you a 360-degree viewing angle.
* **Controls:** Move forward with the **Up Arrow**, move backward with the **Down Arrow**, step right with the **Right Arrow**, step left with the **Left Arrow**. Jump with **Control + Space** (in Item Mode) or **Space** (in Skill Mode). Climb up stairs or surface from underwater using **Page Up**. Climb down stairs or dive underwater using **Page Down**. Sit or stand with **Q**. Rotate 45 degrees using **Shift + Left / Right Arrow**, and rotate 180 degrees using **Shift + Up / Down Arrow**.
* The game also features a few 2D maps.
* You can toggle auto-walk using **Z**.
* **Fall Damage:**
  * Falling from a height of 4 units: Deduces 20 health.
  * Falling from a height of 5 units: Causes bone fractures.
  * The same applies to heights of 6 and 7 units.
* *Note: When coordinates are written as `xxx, 0, 0` in this guide, it means that the **x** coordinate is a variable value depending on your exact horizontal position, while **y** and **z** are 0.*

On any map, you can press **Shift + \`** to see points of interest you can travel to, **Alt + C** to check your current coordinates, **B** to check your current location, and **F** to check the direction you are facing.

### Bone Fracture System:

The game features a mechanic called Bone Fracture.

* When one or more parts of your body are fractured, you can use items in your medical pack to heal.
* The healing process takes some time to fully recover.
* You can check which body parts are fractured and their recovery progress by pressing **O**.

### Experience & Quest Points:

* **Experience Points (XP):** Earned through combat. You can use XP to level up or purchase items in the Experience Points Store.
* **Quest Points:** Earned via the daily lucky spin, delivering fish to the fish restaurant owner, or by completing quests once a quest system is implemented. You can use Quest Points to buy items in the Quest Points Store.

At present, there is only one unofficial quest: **delivering fish to the fish restaurant owner**.

When your character sleeps, your character may dream.

The game uses two main currencies: **coins** (su) and **dollars**.

### Item Mode & Skill Mode:

Keys **1** and **2** correspond to your left and right hands, respectively, and **3** checks what you are holding in both hands.
* **Skill Mode:** When in this mode, keys **1** to **9** are used to punch and kick, and **Space** is used to jump off the ground.
* *Note: You can toggle between the two modes by pressing **Y**.*

### In-Game Animals:

As of now, there are 6 types of animals in the game:

* **Rabbit:** Has 50 health and cannot attack. You can hunt rabbits on the Island.
* **Dog:** Has 2,500 health and can attack. Be careful! Dogs can be found on the Island, in the dog forest on LC Mountain, or in the Coastal City.
* **Bear:** Has 2,500 health. They are quite large and difficult to fight with melee weapons. If possible, fight them using guns. You can hunt bears in the bear cave on LC Mountain.
* **Fox:** Has 1,500 health with moderate attack power. You can hunt foxes in the dark forest near the Coastal City.
* **Wolf:** Has 5,500 health. They have extremely high attack power and fast movement speed. Exercise extreme caution when fighting them.
* **Mutant Wolf:** A mutated wolf with 2,500,000 health. If you are not absolutely sure you can defeat them, stay as far away from them as possible. Remember: stay as far away from them as possible! You can hunt mutant wolves in the dark forest or in the Coastal City.

### Tracking System:

Like most 3D audio games, TW features a tracking system to tell you what objects are near you.

* **Shift + M:** View nearby NPCs and objects.
* **M:** View items lying on the ground on the current map.
* **A:** View nearby animals.
* **Shift + \`:** View points of interest (locations) on the current map that you can travel to.

* **P:** View nearby players.
* **Shift + P:** Open nearby players menu. Press **Enter** on a player to track them.

* **Shift + W:** Toggle auto-tracking for players.

* During rain, lightning will strike. Be careful not to get hit! If you are struck by lightning, a significant amount of health will be deducted.

### Auction System:

The game features an Auction System. This mechanic allows a player to put an item up for auction, and other players can place bids to purchase it.

To start a new auction, you have two methods:

1. **Start an auction via backpack:** Open your backpack (**I**), find the item you want to auction, press **Shift + Enter**, enter the quantity to auction and press **Enter**, then enter the starting/minimum bid (in coins) and press **Enter to begin the auction.
2. **Start an auction via command:** Press **\ (backslash)** to open the command box and type: `auction <item_name> <quantity> <minimum_coins>`.

To place a bid on an active auction, open the command box and type: `bid <amount_of_coins>`. Additionally, you can press shift+b to make bidding easier instead of having to enter complicated orders.

* After a certain period without any new bids, the item will be sold to the highest bidder.
* If no bids are placed or the auction host goes offline, the auction will be canceled.

### Favorite Items:

* **Add to favorites:** Open your backpack, find the item you want to add to your favorite list, and press **Delete**.
* **Navigate favorites:** Cycle through your favorite items list by pressing **Alt + Up / Down Arrow**.
* **Use favorite item:** Press **Backspace** to use the currently selected favorite item.
* **Remove from favorites:** Navigate to the item in your favorite list and press **Shift + Delete**.

* *Warning: Placed cabinets cannot be picked up again!*

### Fire Starter:

* **Function:** Used to ignite fires.
* **Usage:** Hold `fire_starter` and press **Space** to start a fire. Open your backpack, select `wood`, and press **Backspace** to fuel the fire. You can extinguish the fire by pouring water using an `open_water_bottle`.

### Knife:

* **Function:** Used to butcher animal carcasses, craft items, or commit suicide (by cutting your own hand).
* **Usage:** Hold the item to butcher in one hand and the knife in the other, then press **T** to chop.
* *Warning: If your other hand is empty, you will cut your own hand!*
* **Sharpness & Mending:** The knife has a sharpness rating that degrades slightly with each use. When sharpness drops to 0%, the knife becomes useless until sharpened.
  * To sharpen your knife: Hold a `stone` in one hand and the knife in the other. Press **T** repeatedly to sharpen the knife until sharpness reaches its maximum limit of **300%**.
  * You can check your knife's current sharpness by pressing **Alt + \`** (backtick).

### Pot:

* **Function:** Used to hold water (and eventually used to cook noodles in future updates).
* **Usage:** Walk to a water source and press **Space** to scoop water into the pot. Press **Alt + \`** to check the water level. Each pot can hold a maximum of **80 units of water**.
* *Note: Empty water bottles can also collect water in the same manner.*

### Health Booster):

* **Function:** Instantly restores your health to its maximum capacity.
* **Usage:** Hold it in one of your hands and press **Space**.
* *Note: Cannot be used in safe zones.*

### Milk):

* **Usage:** Hold in hand and press **Space** to drink and restore a certain amount of health.

### Energy Drug:

* **Usage:** Hold in hand and press **Space** to drink and restore a substantial amount of energy.

### Chocolate:

* **Usage:** Consumed to restore a minor amount of energy.

### Special Foods & Drinks:

* **Function:** Besides reducing hunger or thirst, these consumables provide permanent stat boosts.
* *Example:* **Soda** and **Cola** permanently increase your maximum health limit.

### Other Foods:

* Sells standard consumables that only reduce hunger and thirst with no additional effects.

### Golden Potion:

* **Function:** A magical potion. Upon consumption, all experience points, coins gained, and resources gathered are **doubled** for a limited time!
* **Usage:** Hold in one of your hands and press **Space** to drink.

## Account Settings:

To open your account settings after connecting to the server, press **F8** and select **Account settings**.

After selecting it, a menu will open with options to change the information related to your account.

You can use the **Up** and **Down Arrow** keys to navigate through these options.

The options available in the **Account settings** menu include:

* **Change name:** This option allows you to change your username.
* **Change password:** This option allows you to change your account password.
* **Set status message:** This option allows you to set a status message that other players (or yourself) can hear when pressing **Control + S** in the players menu, similar to status updates on social media.
* **Trust or untrust a player:** Upon selecting this option, a menu will open listing the players whom you can choose to trust or untrust.
* **Account history:** This data includes information about when your account name or password was changed, and whether the account was previously banned or jailed. You can review your account setting modification history here.
* **Telegram two-factor authentication:** For those of you who have telegram accounts. You can connect to our service to send necessary requests to us anywhere, anytime. In particular, very useful for game managers.
  * Set Telegram ID: Please set the ID address of your telegram account so that the server can connect you to the service.
  * Open Telegram bot to link my ID: you can bind to the service if you have set the correct ID address.
  * Enable Telegram 2FA: You can turn it on to verify that this is your account. Once authentication is complete, you can disable this option.
  * Remove Telegram ID: If you change to another telegram account, this option will delete the ID addresses you have set up.

## Bug Reporting & Feature Suggestion:

To report an existing bug in the game or suggest a new feature to the developers after connecting to the server, press **F8** and select **Support and contact**.

After selecting **Support and contact**, a menu will open with the following options:

* **View your support tickets:** This option allows you to review the support tickets you have created to suggest new features.
* **Create a support ticket:** This option allows you to create a new support ticket specifically for proposing new features.
* **Report a bug:** If you discover a bug in the game, you can use this option to report it to the developers.
* **View the reports you have submitted and the status of the reports:** This option helps you review the bug reports you have submitted and track their resolution status.
* **View game staff and ranks:** This option allows you to view the list of game developers and administrators along with their ranks.
